# Cybersecurity Homelab

## Overview

This repository documents the development of my personal cybersecurity homelab, built to gain hands-on experience with virtualization, Linux administration, networking, system administration and defensive security.

Rather than following tutorials without understanding them, the aim of this project is to understand why technologies are used and how they work together.

## Current Hardware

* Dell OptiPlex 3040 SFF
* Intel Core i5-6500
* 16 GB RAM
* 256 GB SSD
* TP-Link 5-Port Gigabit Switch

## Current Software

* Proxmox VE
* Ubuntu Server 26.04 LTS

## Learning Objectives

* Linux Administration
* Networking Fundamentals
* Docker
* Pi-hole
* Plex
* Windows Server
* Active Directory
* SIEM (Wazuh)
* Virtualization
* System Hardening
